YouTubeUnstoppablePlayer
========================

「YouTube動画の一時停止を不可にしてコントロールバーを非表示にする仕組み （HTML、IFrameプレーヤーAPI（JavaScript）、CSS）」

<iframe>タグの埋め込みコードをつかったHTML5形式の動画プレーヤーです。
